== Description ==

= Overview =
This plugin is for picking images from the [Kuvat.fi](https://kuvat.fi) photo sharing service and downloading them into your WordPress media library.

Kuvat.fi creates a great looking photo site from your photos. It's easy to use and you can use it with your web browser, tablets and mobile devices. We're based in Finland and have been running for 15 years.

== Installation ==

1. Upload `kuvatfi-picker` folder with its contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Get your [Kuvat.fi API Key](https://tuki.kuvat.fi/artikkelit/kuvien-haku-wordpressiin-valitsin-lisaosalla/)
1. Enter the API Key in the plugin's settings

== Usage ==

After entering your API Key, a tab called Kuvat.fi should appear in the WordPress media picker window, next to Upload Files and Media Library. The files picked with this plugin will be downloaded as local copies from Kuvat.fi into your WordPress media library.
